# データ型攻撃パターン集

全入力フィールドに対して以下のパターンを体系的に適用する。

## 数値入力

| パターン | テスト値 | 検証内容 |
|---------|---------|---------|
| ゼロ | 0 | ゼロ除算、初期値との区別 |
| 負数 | -1, -999 | 符号処理 |
| 最大値境界 | 32767, 32768 (int16) | オーバーフロー |
| 最大値境界 | 2147483647, 2147483648 (int32) | オーバーフロー |
| 最大値境界 | 9007199254740991 (JS MAX_SAFE_INTEGER) | 精度損失 |
| 小数 | 0.1, 0.01, 1.005 | 浮動小数点精度 |
| 科学記法 | 1E-16, 1E+308 | パース処理 |
| 非数値 | NaN, Infinity, -Infinity | 特殊値処理 |
| カンマ区切り | 1,000 / 1.000 (EU) | ロケール |

## 文字列入力

| パターン | テスト値 | 検証内容 |
|---------|---------|---------|
| 空文字 | "" | 必須チェック |
| 1文字 | "a" | 最小長 |
| 長文字列 | 255文字, 256文字, 1024文字, 10000文字 | 上限処理 |
| 特殊文字 | `!@#$%^&*(){}[]|\:;"'<>,.?/~` | エスケープ |
| SQL injection | `' OR '1'='1`, `'; DROP TABLE users; --` | サニタイズ |
| XSS | `<script>alert('xss')</script>` | エスケープ |
| HTMLタグ | `<b>bold</b>`, `<img src=x onerror=alert(1)>` | サニタイズ |
| 改行 | `line1\nline2`, `line1\r\nline2` | 改行処理 |
| Unicode | `漢字`, `العربية`, `한국어` | マルチバイト |
| 絵文字 | `👨‍👩‍👧‍👦`, `🏳️‍🌈` | サロゲートペア |
| NULL文字 | `\0`, `%00` | ヌルバイト |
| パストラバーサル | `../../etc/passwd`, `..\..\windows` | パス検証 |
| スペース | 先頭/末尾スペース, 全角スペース | トリム処理 |
| 制御文字 | タブ, バックスペース, ESC | 非表示文字 |

## 日付・時刻入力

| パターン | テスト値 | 検証内容 |
|---------|---------|---------|
| 不正日付 | 2/30, 4/31, 13/1 | バリデーション |
| 閏年 | 2/29 (閏年), 2/29 (非閏年) | 閏年判定 |
| 境界日付 | 1970-01-01, 2038-01-19, 9999-12-31 | エポック境界 |
| フォーマット混在 | MM/DD/YYYY vs DD.MM.YYYY | ロケール |
| タイムゾーン | UTC, JST(+9), EST(-5) | TZ変換 |
| 夏時間 | 3月第2日曜, 11月第1日曜 (US) | DST切替 |
| ISO 8601 | 2026-03-15T10:30:00Z | フォーマット |

## ファイル入力

| パターン | テスト値 | 検証内容 |
|---------|---------|---------|
| 0バイト | 空ファイル | 空ファイル処理 |
| 巨大ファイル | 100MB+ | サイズ制限 |
| 不正拡張子 | .exe → .jpg | MIME検証 |
| 長いファイル名 | 255文字+ | ファイルシステム制限 |
| 特殊文字名 | `file name (1).txt` | パス処理 |
| 二重拡張子 | `file.jpg.exe` | 拡張子チェック |
