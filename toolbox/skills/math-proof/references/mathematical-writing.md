# 数学的記述の明晰性ガイド

## 証明の構造

### 良い証明の要件

1. **完全性**: すべてのステップに根拠がある
2. **正確性**: 各ステップが論理的に正しい
3. **明晰性**: 読者が各ステップを追える
4. **簡潔性**: 不要な冗長さがない
5. **動機付け**: なぜそのアプローチを取るのかが分かる

### 証明の構成パターン

```
**定理 N.M** (定理名).
[定理の正確な主張]

**証明.**
[証明の概略・戦略の説明（任意だが推奨）]

[ステップ 1: 設定・仮定の導入]

[ステップ 2-N: 論理展開]

[最終ステップ: 結論]
∎
```

## 記述のベストプラクティス

### やるべきこと

- **変数の導入時に型を明示**: 「$\varepsilon > 0$ を任意にとる」
- **定理の使用時に名前を記載**: 「中間値の定理より...」
- **論理の流れを示す接続詞**: 「したがって」「一方」「ここで」
- **重要な等式変形には根拠を**: 「$= ... $（三角不等式より）」
- **証明の冒頭で戦略を述べる**: 「背理法で示す」「帰納法で証明する」

### 避けるべきこと

- **「明らか」「自明」の多用**: 省略されたステップは誤りの温床
- **記号の乱用**: 同じ記号を異なる意味で使わない
- **暗黙の量化子**: 「$f(x) > 0$」→「すべての $x$ で？ある $x$ で？」
- **証明の途中での定義変更**: 一度定義した記号は変えない
- **長すぎる一文**: 複数のステップを一文に詰め込まない

## 定義の書き方

```
**定義** (コンパクト).
位相空間 $X$ が**コンパクト**であるとは、
$X$ の任意の開被覆が有限部分被覆を持つことをいう。

すなわち、$\{U_\alpha\}_{\alpha \in A}$ が $X$ の開被覆
（$X = \bigcup_{\alpha \in A} U_\alpha$、各 $U_\alpha$ は開集合）ならば、
有限個の $\alpha_1, \ldots, \alpha_n \in A$ が存在して
$X = \bigcup_{i=1}^{n} U_{\alpha_i}$ が成り立つ。
```

## 反例の提示

```
**反例** (連続関数の逆は必ずしも連続でない).
$f: [0, 2\pi) \to S^1$, $f(t) = e^{it}$ は連続な全単射だが、
$f^{-1}$ は不連続。実際、$S^1$ 上で $1$ に収束する点列
$\{e^{i(2\pi - 1/n)}\}$ の逆像は $\{2\pi - 1/n\}$ であり、
$f^{-1}(1) = 0$ に収束しない。
```

## チェックリスト

証明を書き終えたら確認:

- [ ] すべての変数に型宣言があるか
- [ ] 使用する定理の前提条件が満たされているか
- [ ] 場合分けが網羅的か
- [ ] 帰納法の基底段階を忘れていないか
- [ ] 「存在する」で取った元を正しく使っているか
- [ ] 結論が本当に示したい主張と一致しているか
- [ ] 読者の知識レベルに適した詳細度か
