# 批判的分析フレームワーク

## 分析の3層構造

### Layer 1: 記述（Description）

論文が「何を述べているか」を正確に把握する。

- **研究目的**: 何を明らかにしようとしているか
- **RQ（Research Questions）**: 具体的な研究課題
- **仮説**: テスト対象の仮説（あれば）
- **方法**: どのように調査したか
- **結果**: 何が見つかったか
- **結論**: 著者は何を主張しているか

### Layer 2: 分析（Analysis）

論文の「論理構造と整合性」を評価する。

- **主張とエビデンスの対応**: 各主張は十分なエビデンスで支持されているか
- **論理の連鎖**: 前提→推論→結論の流れは妥当か
- **内的整合性**: 論文内で矛盾がないか
- **方法論的妥当性**: 方法は目的に適しているか
- **統計的妥当性**: 分析手法と解釈は正しいか

### Layer 3: 評価（Evaluation）

論文の「貢献と限界」を判断する。

- **新規性**: 既存研究に対する貢献は何か
- **一般化可能性**: 結果はどこまで一般化できるか
- **再現可能性**: 再現に必要な情報は十分か
- **影響**: 分野への影響は何か
- **限界**: 著者が認識している/していない限界

## 批判的読解の問い

### 主張に対して

1. この主張は明確に述べられているか？
2. この主張を支持するエビデンスは何か？
3. エビデンスの質は十分か？
4. 代替解釈は考えられるか？
5. この主張の一般化の範囲は？

### 方法に対して

1. 研究デザインは RQ に適切か？
2. サンプルは母集団を代表しているか？
3. 測定方法は信頼性・妥当性があるか？
4. 交絡変数は適切に統制されているか？
5. 倫理的配慮は十分か？

### 結果に対して

1. 結果は明確に報告されているか？
2. 統計的有意性と実質的有意性は区別されているか？
3. 効果量は報告されているか？
4. ネガティブな結果も報告されているか？
5. 結果の解釈は妥当か？

### 結論に対して

1. 結論はデータから論理的に導かれるか？
2. 結論は主張の強さに見合っているか？
3. 限界は適切に認識されているか？
4. 今後の研究への示唆は妥当か？

## CRAAP テスト

| 基準 | 評価ポイント |
|------|------------|
| **C**urrency（新しさ） | 出版年、データの新しさ、分野での陳腐化速度 |
| **R**elevance（関連性） | RQ への適合度、対象読者 |
| **A**uthority（権威） | 著者の専門性、所属機関、引用実績 |
| **A**ccuracy（正確性） | データ・方法の正確さ、査読の有無 |
| **P**urpose（目的） | 研究の動機、利益相反、バイアスの可能性 |
