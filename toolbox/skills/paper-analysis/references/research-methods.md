# 研究手法の評価基準

## 研究デザインの分類

### 実験研究

| デザイン | 特徴 | 強み | 弱み |
|----------|------|------|------|
| RCT（無作為化比較試験） | 無作為割付、対照群 | 因果推論に最適 | コスト高、倫理的制約 |
| 準実験 | 対照群あり、無作為化なし | 現実的な場面で実施可能 | 選択バイアスの可能性 |
| 前後比較 | 介入前後の比較 | 実施容易 | 時間経過の影響を排除できない |

### 観察研究

| デザイン | 特徴 | 強み | 弱み |
|----------|------|------|------|
| コホート研究 | 前向き追跡 | 時間的順序が明確 | 脱落、長期間必要 |
| ケースコントロール | 後ろ向き比較 | 稀な疾患に有効 | 想起バイアス |
| 横断研究 | 一時点の調査 | 迅速、安価 | 因果関係不明 |

### 質的研究

| 手法 | 特徴 | 適用場面 |
|------|------|----------|
| インタビュー | 深い理解 | 経験・認識の探索 |
| 参与観察 | 自然な文脈 | 文化・実践の理解 |
| グラウンデッド・セオリー | 理論構築 | 未知の領域 |
| 事例研究 | 詳細な分析 | 特殊事例の理解 |

## 妥当性の評価

### 内的妥当性（Internal Validity）

因果関係の推論がどの程度信頼できるか。

**脅威**:
- 歴史効果（研究期間中の外部イベント）
- 成熟効果（時間経過による自然変化）
- テスト効果（事前テストの影響）
- 計測器変動（測定の変化）
- 統計的回帰（平均への回帰）
- 選択バイアス（グループ間の系統的差異）
- 脱落（参加者の離脱）

### 外的妥当性（External Validity）

結果がどの程度一般化できるか。

**評価ポイント**:
- 母集団への一般化（サンプルの代表性）
- 環境への一般化（ラボ vs 実世界）
- 時間への一般化（結果の持続性）
- 測定への一般化（異なる測定方法で同じ結果か）

### 構成概念妥当性（Construct Validity）

測定が意図した概念を正しく捉えているか。

**評価ポイント**:
- 操作的定義の妥当性
- 収束的妥当性（同じ概念の別測定との相関）
- 弁別的妥当性（異なる概念との低相関）

## サンプリング評価基準

| 基準 | 確認事項 |
|------|----------|
| サンプルサイズ | 検定力分析は行われたか？十分なサイズか？ |
| サンプリング方法 | 無作為？便宜的？層化？ |
| 包含/除外基準 | 明確に定義されているか？ |
| 代表性 | 母集団を適切に反映しているか？ |
| 脱落 | 脱落率は？系統的脱落はないか？ |

## 測定の評価基準

| 基準 | 確認事項 |
|------|----------|
| 信頼性（Reliability） | テスト再テスト、内的一貫性、評価者間一致 |
| 妥当性（Validity） | 内容、基準関連、構成概念 |
| 感度（Sensitivity） | 変化を検出できるか |
| 特異度（Specificity） | 正しく区別できるか |
