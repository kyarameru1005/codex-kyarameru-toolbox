# エビデンスの階層

## エビデンスピラミッド

```
        ┌─────────────┐
        │  系統的レビュー  │  ← 最高レベル
        │  メタ分析     │
        ├─────────────┤
        │   RCT       │
        ├─────────────┤
        │  コホート研究   │
        ├─────────────┤
        │ ケースコントロール │
        ├─────────────┤
        │  横断研究     │
        ├─────────────┤
        │  症例報告     │
        ├─────────────┤
        │  専門家の意見   │  ← 最低レベル
        └─────────────┘
```

## 各レベルの詳細

### Level 1: 系統的レビュー・メタ分析

| 特徴 | 詳細 |
|------|------|
| 定義 | 複数の研究を体系的に統合 |
| 強み | 最も包括的、バイアス低減 |
| 注意点 | 含まれる研究の質に依存、出版バイアスの影響 |
| 評価基準 | PRISMA ガイドライン準拠、異質性の評価、感度分析 |

### Level 2: 無作為化比較試験（RCT）

| 特徴 | 詳細 |
|------|------|
| 定義 | 無作為割付による介入研究 |
| 強み | 因果推論に最適 |
| 注意点 | 外的妥当性の制限、倫理的制約 |
| 評価基準 | CONSORT ガイドライン、ITT 分析、盲検化 |

### Level 3: コホート研究

| 特徴 | 詳細 |
|------|------|
| 定義 | 曝露群と非曝露群の前向き追跡 |
| 強み | 時間的順序が明確、稀な曝露に有用 |
| 注意点 | 交絡因子、長期間・高コスト |
| 評価基準 | STROBE ガイドライン、追跡率、交絡の統制 |

### Level 4: ケースコントロール研究

| 特徴 | 詳細 |
|------|------|
| 定義 | 症例群と対照群の後ろ向き比較 |
| 強み | 稀な疾患に有効、迅速・安価 |
| 注意点 | 想起バイアス、対照群の選択 |
| 評価基準 | STROBE ガイドライン、マッチング、曝露の測定 |

### Level 5: 横断研究

| 特徴 | 詳細 |
|------|------|
| 定義 | 一時点での調査 |
| 強み | 迅速、有病率の推定 |
| 注意点 | 因果関係不明、逆因果の可能性 |

### Level 6: 症例報告・症例シリーズ

| 特徴 | 詳細 |
|------|------|
| 定義 | 個別症例の詳細な記述 |
| 強み | 新しい現象の発見、仮説生成 |
| 注意点 | 一般化不可、バイアス大 |

### Level 7: 専門家の意見

| 特徴 | 詳細 |
|------|------|
| 定義 | エビデンスに基づかない専門家の見解 |
| 強み | 未研究領域での指針 |
| 注意点 | 主観的、バイアスの影響大 |

## GRADE システム

エビデンスの確実性を4段階で評価:

| レベル | 意味 | 格上げ要因 | 格下げ要因 |
|--------|------|-----------|-----------|
| ⊕⊕⊕⊕ 高 | 効果推定に高い確信 | - | - |
| ⊕⊕⊕○ 中 | 効果推定にかなりの確信 | 大きな効果量 | バイアスリスク |
| ⊕⊕○○ 低 | 確信は限定的 | 用量反応関係 | 非一貫性 |
| ⊕○○○ 非常に低 | ほとんど確信なし | 交絡因子が効果を弱める方向 | 非直接性、不精確 |

## 評価時の問い

1. **エビデンスの種類は何か？**（デザインレベル）
2. **バイアスのリスクは？**（内的妥当性）
3. **結果の一貫性は？**（複数研究間の整合性）
4. **直接性は？**（RQ への直接的な回答か）
5. **精確性は？**（信頼区間の幅）
6. **出版バイアスの可能性は？**
