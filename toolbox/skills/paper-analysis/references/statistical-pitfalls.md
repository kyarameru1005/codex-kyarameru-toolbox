# 統計的誤謬・落とし穴

## p値に関する誤謬

### 1. p値の誤解釈

**誤り**: 「p = 0.03 は、帰無仮説が正しい確率が3%」
**正解**: 「帰無仮説が正しい場合に、このデータ（またはより極端なデータ）が得られる確率が3%」

### 2. 統計的有意性 ≠ 実質的有意性

- p < 0.05 でも効果が微小な場合がある
- 大サンプルでは些細な差も有意になる
- **必ず効果量（Cohen's d, η², r²等）を確認**

### 3. 有意でない ≠ 効果がない

- 検定力不足の可能性
- 「差がないことの証明」には等価性検定が必要

## 多重比較の問題

### Family-wise Error Rate

- $k$ 回の独立な検定で少なくとも1回偽陽性が出る確率: $1 - (1 - \alpha)^k$
- 20回の検定 → 偽陽性の確率 ≈ 64%

### 補正方法

| 方法 | 特徴 |
|------|------|
| Bonferroni | 保守的、$\alpha/k$ |
| Holm | Bonferroniより検出力高 |
| Benjamini-Hochberg | FDR制御、探索的分析向き |
| Tukey HSD | 全対比較向き |

## p-hacking（データドレッジング）

### 兆候

- 多数の検定のうち有意なもののみ報告
- 外れ値の選択的除外
- サンプルサイズの途中変更
- 共変量の選択的投入
- サブグループ分析の事後的実施

### 確認方法

- p値の分布が0.05付近に集中していないか
- 事前登録との一致を確認
- すべての分析結果が報告されているか

## HARKing

**Hypothesizing After the Results are Known**

- 結果を見てから仮説を立て、あたかも事前仮説であったかのように報告
- 確認的研究と探索的研究の区別がない

## 基本的な統計の落とし穴

### 相関と因果

- 相関は因果を含意しない
- 交絡変数の可能性を常に考慮
- 因果推論には実験デザインまたは因果推論手法が必要

### Simpson のパラドックス

- 全体の傾向とサブグループの傾向が逆転する現象
- 層化分析の重要性

### 生態学的誤謬

- 集団レベルの相関を個人レベルに適用する誤り

### 前提条件の不充足

| 検定 | 主な前提条件 |
|------|------------|
| t検定 | 正規性、等分散性 |
| ANOVA | 正規性、等分散性、独立性 |
| 回帰分析 | 線形性、正規性、等分散性、独立性 |
| カイ二乗検定 | 期待度数 ≥ 5 |

### サンプルサイズの問題

| 問題 | 影響 |
|------|------|
| 過小 | 検定力不足、不安定な推定 |
| 過大 | 些細な差も有意に、過適合 |
