# 論理的誤謬一覧

## 形式的誤謬（Formal Fallacies）

### 前件否定の誤謬（Denying the Antecedent）
- **形式**: $P \implies Q, \neg P \vdash \neg Q$（無効）
- **例**: 「雨なら地面が濡れる。雨でない。∴地面は濡れていない」
- **問題**: スプリンクラーで濡れる可能性

### 後件肯定の誤謬（Affirming the Consequent）
- **形式**: $P \implies Q, Q \vdash P$（無効）
- **例**: 「薬が効くなら症状が改善する。症状が改善した。∴薬が効いた」
- **問題**: 自然治癒の可能性

### 量化子の順序誤り
- $\forall x \exists y: R(x,y)$ と $\exists y \forall x: R(x,y)$ は異なる
- **例**: 「各学生に指導教員がいる」≠「全学生に共通の指導教員がいる」

## 非形式的誤謬（Informal Fallacies）

### 論証の誤り

| 誤謬 | 説明 | 学術論文での例 |
|------|------|--------------|
| 循環論法 | 結論を前提に含む | 定義の中に定義対象を使用 |
| 藁人形論法 | 相手の主張を歪曲 | 先行研究の主張を過度に単純化 |
| 無知に訴える論証 | 反証がない→正しい | 「否定するエビデンスがないので正しい」 |
| 権威に訴える論証 | 権威者が言った→正しい | 分野外の権威の意見を根拠に |
| 二分法の誤り | 2択しかないと仮定 | 「AかBのどちらか」（Cもあり得る） |
| 滑り坂論法 | 連鎖的帰結を誇張 | 「この結果を認めると〜になる」 |

### 因果推論の誤り

| 誤謬 | 説明 | 学術論文での例 |
|------|------|--------------|
| 前後即因果 | 時間的前後→因果 | 「介入後に改善→介入が原因」 |
| 相関即因果 | 相関→因果 | 「A と B が相関→A が B を引き起こす」 |
| 逆因果 | 因果の方向が逆 | 「運動する人は健康→運動が健康を作る」（逆かも） |
| 第三変数 | 交絡因子の無視 | SES が両方に影響しているのに無視 |

### 一般化の誤り

| 誤謬 | 説明 | 学術論文での例 |
|------|------|--------------|
| 早まった一般化 | 不十分なサンプルから一般化 | n=20 の結果を母集団全体に適用 |
| 生態学的誤謬 | 集団→個人への不当な推論 | 国レベルの相関を個人に適用 |
| 例外の無視 | 反例を無視 | 理論に合わないデータを外れ値として除外 |
| チェリーピッキング | 都合の良いデータのみ選択 | 仮説を支持するデータのみ報告 |

### 曖昧さの誤り

| 誤謬 | 説明 | 学術論文での例 |
|------|------|--------------|
| 多義語の誤謬 | 同じ語を異なる意味で使用 | 「有意」を統計的/実質的に混用 |
| 合成の誤謬 | 部分→全体への不当な推論 | 個々の薬が安全→併用も安全 |
| 分割の誤謬 | 全体→部分への不当な推論 | チーム全体が優秀→各メンバーも優秀 |

## 科学論文特有の問題

### テキサスの名射手

- データを見てから仮説を設定（HARKing）
- 偶然のパターンに意味を見出す

### 追認バイアス（Confirmation Bias）

- 仮説を支持するデータに注目
- 反証データを軽視・無視

### 出版バイアス

- 有意な結果のみ出版される傾向
- ファイルドロワー問題

### 基底率の無視

- 事前確率を考慮しない
- ベイズの定理を無視した解釈
