# 数理構造の分野横断対応表

数理構造が複数分野で現れるパターンの網羅的対応表。
探索時に「この構造は他分野ではどう呼ばれているか」を素早く引くためのリファレンス。

---

## 1. 微分方程式系

| 分野 | 具体的な系 | 共通構造 |
|------|-----------|---------|
| 古典力学 | Newton 方程式 | 2階ODE、ハミルトニアン形式 |
| 電磁気学 | Maxwell 方程式 | 連立PDE、ゲージ対称性 |
| 流体力学 | Navier-Stokes 方程式 | 非線形PDE、乱流 |
| 生態学 | Lotka-Volterra 方程式 | 非線形ODE系、動的平衡 |
| 金融工学 | Black-Scholes 方程式 | 放物型PDE、確率微分方程式 |
| 疫学 | SIR モデル | コンパートメントODE |
| 神経科学 | Hodgkin-Huxley モデル | 非線形ODE、興奮系 |
| 化学 | 反応速度論 | 質量作用の法則、ODE系 |

**共通する解析ツール**: 安定性解析、分岐理論、数値解法、摂動法

## 2. 確率・統計モデル

| 分野 | 応用 | 共通構造 |
|------|------|---------|
| 統計力学 | Boltzmann 分布 | 指数族、分配関数 |
| 自然言語処理 | 言語モデル | マルコフ連鎖、条件付き確率 |
| 金融工学 | リスク評価 | 確率過程、マルチンゲール |
| 遺伝学 | 集団遺伝学 | Wright-Fisher モデル、拡散過程 |
| 通信理論 | 誤り訂正 | 確率的復号、ベイズ推論 |
| 機械学習 | ベイズ推論 | 事後分布、MCMC |

**共通する解析ツール**: 大数の法則、中心極限定理、大偏差原理、エルゴード理論

## 3. 情報理論

| 分野 | エントロピーの解釈 | 核心的な量 |
|------|-------------------|-----------|
| 熱力学 | 熱力学的エントロピー $S = k_B \ln \Omega$ | 自由エネルギー |
| 通信理論 | Shannon エントロピー $H = -\sum p \log p$ | チャネル容量 |
| 機械学習 | 交差エントロピー損失 | KL ダイバージェンス |
| 統計学 | Fisher 情報量 | Cramer-Rao 下界 |
| 計算理論 | Kolmogorov 複雑性 | 最小記述長 |
| 量子情報 | von Neumann エントロピー | 量子もつれ |

**重要な橋渡し**: Jaynes の最大エントロピー原理 — 統計力学とベイズ推論の統一

## 使い方

1. 探索対象の現象が**どのカテゴリに属するか**を特定
2. 同じカテゴリの**他分野での出現**を確認
3. **共通する解析ツール**が転用可能か検討
4. 必要に応じて **structural-analogies.md** でメタ原理を参照
