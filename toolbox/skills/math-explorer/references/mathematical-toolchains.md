# 分野別の数理ツールチェーン

各応用分野で頻出する数理ツールの「ツールチェーン」一覧。
ある分野のツールが他分野で転用可能かを判断するためのリファレンス。

## 各分野の詳細ツールチェーン

### 統計力学

```
ミクロ状態の列挙
  → 分配関数 Z = Σ exp(-βE)
    → 自由エネルギー F = -kT ln Z
      → 熱力学的諸量（エントロピー、比熱等）
        → 相転移の分類（Ehrenfest / Landau）
          → 繰り込み群（臨界指数、普遍性クラス）
```

**ML への転用**: エネルギー関数→損失関数、温度→学習率、相転移→学習の臨界現象

### 情報理論

```
確率分布 p(x)
  → Shannon エントロピー H(X) = -Σ p log p
    → 相互情報量 I(X;Y) = H(X) - H(X|Y)
      → チャネル容量 C = max I(X;Y)
        → レート歪み理論 R(D)
          → ソースコーディング / チャネルコーディング
```

**ML への転用**: 交差エントロピー損失、情報ボトルネック法、変分下界

### 微分幾何

```
多様体 M
  → 接空間 TM、余接空間 T*M
    → 計量テンソル g（リーマン計量）
      → 接続 ∇（レヴィ・チヴィタ接続）
        → 曲率テンソル R
          → 測地線（最短経路）
            → Gauss-Bonnet 定理（局所 → 大域）
```

**最適化への転用**: パラメータ多様体上の自然勾配降下法

### 関数解析

```
ベクトル空間
  → ノルム空間 → バナッハ空間
    → 内積空間 → ヒルベルト空間
      → 線形作用素 → 有界/非有界作用素
        → スペクトル理論
          → 固有値分解 / SVD
            → 関数微積分（作用素の関数）
```

**ML への転用**: カーネル法（RKHS）、スペクトルクラスタリング、ニューラル正接カーネル

### 代数的位相幾何

```
位相空間 X
  → 単体複体 / CW 複体
    → チェイン複体 C_*
      → ホモロジー群 H_*(X)
        → ベッチ数 β_k（k次元の「穴」の数）
          → オイラー標数 χ = Σ(-1)^k β_k
            → パーシステントホモロジー（データ解析）
```

**データ解析への転用**: TDA（Topological Data Analysis）、Mapper アルゴリズム

### 確率論

```
確率空間 (Ω, F, P)
  → 確率変数 X: Ω → R
    → 分布関数、特性関数
      → 確率過程 {X_t}
        → マルコフ過程 / マルチンゲール
          → 確率微分方程式（Itô / Stratonovich）
            → 大偏差原理、集中不等式
```

**金融工学への転用**: 確率微分方程式→デリバティブ価格付け、マルチンゲール→リスク中立測度

### 圏論

```
対象と射
  → 圏 C
    → 関手 F: C → D
      → 自然変換 η: F ⇒ G
        → 随伴関手 F ⊣ G
          → モナド T = GF
            → 極限 / 余極限
              → Yoneda 補題
```

**プログラミングへの転用**: モナド→副作用の制御（Haskell）、関手→コンテナ型

### 制御理論

```
システム ẋ = f(x, u)
  → 可制御性 / 可観測性
    → 状態フィードバック u = Kx
      → 安定性解析（リアプノフ関数）
        → 最適制御（Pontryagin の最大原理 / HJB 方程式）
          → ロバスト制御（H∞制御）
            → 適応制御
```

**強化学習への転用**: 最適制御→ベルマン方程式→Q学習、リアプノフ→安全な学習

### ゲーム理論

```
プレイヤー、戦略、利得関数
  → 支配戦略 / Nash 均衡
    → 混合戦略均衡
      → 繰り返しゲーム（Folk 定理）
        → メカニズムデザイン（逆問題）
          → オークション理論
            → 進化的安定戦略（ESS）
```

**AI への転用**: GAN（二人ゼロ和ゲーム）、マルチエージェント強化学習、RLHF

### 計算複雑性

```
計算問題
  → 決定問題 / 探索問題
    → 時間複雑性クラス（P, NP, PSPACE...）
      → 帰着（多項式時間）
        → NP完全性 / NP困難性
          → 近似アルゴリズム / 近似困難性
            → パラメータ化複雑性（FPT）
```

**ML理論への転用**: 学習の計算困難性、サンプル複雑性、PAC学習

## 分野間の接続パターン

### よく見られる接続

```
統計力学 ←→ 情報理論     (エントロピー、自由エネルギー)
確率論   ←→ 金融工学     (確率微分方程式、マルチンゲール)
微分幾何 ←→ 情報理論     (情報幾何)
関数解析 ←→ 量子力学     (ヒルベルト空間、作用素)
圏論     ←→ プログラミング (型理論、モナド)
制御理論 ←→ 強化学習     (最適制御、ベルマン方程式)
ゲーム理論 ←→ 進化生物学  (ESS、レプリケータ方程式)
代数的位相幾何 ←→ データ科学 (TDA、パーシステントホモロジー)
```

### 使い方

1. 解決したい問題が属する分野のツールチェーンを確認
2. ツールチェーン上で「最も近い接続先」を特定
3. 接続先分野のツールが転用可能か検討
4. 必要に応じて structural-analogies.md のメタ原理を参照
