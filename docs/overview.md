# Overview

このリポジトリは、`~/.codex` へ配備する個人用ツールボックスを管理する。

## 目的

- `toolbox/` 配下のスキル、エージェント、フック、プラグイン資材を安全に配備する。
- AI コーディングエージェント運用のルール、検証手順、ハーネスを同一リポジトリで維持する。
- 研究活動としての記録と、実運用に必要な手順書を分離して管理する。

## 入口文書

- `AGENTS.md`: リポジトリ運用ルールの正本。
- `README.md`: セットアップ、インストール、検証コマンドの入口。
- `docs/repository-layout.md`: ディレクトリ責務と Git 管理方針。
- `docs/research.md`: 研究計画と背景。
- `docs/harness-spec.md`: ハーネス検証の仕様。

## 標準構造での位置づけ

- `docs/overview.md`: リポジトリ全体像の入口。
- `docs/requirements.md`: 維持すべき要求の整理。
- `docs/architecture.md`: 主要ディレクトリと運用構造の説明。
- `docs/understanding-check.md`: 変更前後に確認すべき理解ポイント。

既存の詳細文書は残し、上記 4 文書は案内板として使う。
