# muse-modern

クセを抑えた現代的で清潔な見た目を与える系統。
`muse-interface`（共通の UI 構造設計）の上に、モダンの見た目トーンを与える。

## この系統の狙い / 与える印象

- 清潔・整然・信頼感。情報がすっと読める
- 派手さより「過不足のなさ」で品よく見せる

## 配色・トーン

- ベースは白〜淡いグレー、テキストは濃いグレー（純黒は避ける）
- アクセントは1色に絞る（彩度は中程度）
- コントラストは可読性優先（本文は AA 以上）

## タイポグラフィ

- サンセリフ中心。字種は2つまで
- 見出しと本文の差はサイズと太さで付ける（装飾に頼らない）

## レイアウト・余白

- 余白は多め。要素を詰め込まない
- グリッド整列と一貫した間隔スケール

## 装飾・質感・モーション

- フラット寄り。影は薄く小さく
- 角丸は控えめに統一。モーションは短く控えめ（説明的な動きのみ）

## 避けること（アンチパターン）

- 多色・強い装飾・過度なグラデーション
- 余白を削った詰め込みレイアウト

## 出力（Hephaestus に渡す見た目指針）

- 配色（ベース/テキスト/アクセント1色）の方向
- 余白スケールと角丸・影の強さ
- タイポの字種数と見出し/本文の差の付け方
