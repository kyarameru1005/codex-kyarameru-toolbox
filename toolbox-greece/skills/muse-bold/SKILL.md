# muse-bold

強いコントラストと太さで「かっこよさ」を出す系統。
`muse-interface`（共通の UI 構造設計）の上に、大胆の見た目トーンを与える。

## この系統の狙い / 与える印象

- 力強い・先鋭・勢い。第一印象でインパクトを残す
- 主役を大きく見せ、メリハリで引き込む

## 配色・トーン

- 黒/濃色基調＋鮮やかなアクセント1〜2色
- 高コントラスト。差を強く取る（彩度・明度の落差）

## タイポグラフィ

- 太い・大きい見出し（ディスプレイ書体可）
- 見出しと本文のサイズ差を大きく取る

## レイアウト・余白

- 大きな要素・全幅のブロック・大胆な余白の偏り
- 対角・ずらし配置など動きのある構図も可

## 装飾・質感・モーション

- 強い影・縁取り・グラデを意図的に使う
- モーションは速くキレのある動き（ホバー/出現で印象付け）

## 避けること（アンチパターン）

- 低コントラストで弱い印象
- 要素を均一に並べて平坦に見せる

## 出力（Hephaestus に渡す見た目指針）

- 黒基調＋アクセント色とコントラストの強さ
- 見出しの太さ・大きさと本文との差
- モーションの速さ・キレと使いどころ
