# Zeus Start Prompt

Zeus を入口にして `toolbox-greece` のオーケストレーションを始めるためのテンプレート。
必要に応じて内容を書き換えて使う。

```text
Zeusを使ってこの作業を進めてください。

目的:
- <何を達成したいか>

前提:
- <対象リポジトリや技術スタック>
- <既知の制約や守るべき条件>

完了条件:
- <何ができたら終わりか>

進め方:
- まず Zeus が必要な支援を判断する
- 関連ファイルや影響範囲が不明なら Hermes を使う
- 設計判断が必要なら Daedalus を使う
- 実装が必要なら Hephaestus を使う
- 実装後は Athena または Themis で確認する
- 必要なら Apollo または Chronos を使う

出してほしいもの:
- 最初の判断
- 進行計画
- 必要ならサブエージェント利用方針
```

## 短い版

```text
Zeusを使って進めてください。必要なら Hermes で調査し、Daedalus で設計し、Hephaestus で実装し、Athena または Themis で確認してください。
```
